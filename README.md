 
## Introduction 
 
AnyPay is a fintech banking application built with Next.js, TypeScript, Tailwind CSS, Appwrite, Plaid, and Dwolla. It allows users to connect multiple bank accounts, view transactions in real-time, and transfer funds securely. 
 
## Tech Stack 
 
- Next.js 
- TypeScript 
- Tailwind CSS 
- Appwrite 
- Plaid 
- Dwolla 
- React Hook Form 
- Zod 
- Chart.js 
- ShadCN 
 
## Features 
 
- **Authentication**: Secure SSR authentication with validations and authorization 
- **Connect Banks**: Integrates with Plaid for multiple bank account linking 
- **Home Page**: Overview of user account with total balance, recent transactions, spending categories 
- **My Banks**: Complete list of connected banks with balances and account details 
- **Transaction History**: Pagination and filtering for transaction history 
- **Real-time Updates**: Reflects changes across all pages upon connecting new accounts 
- **Funds Transfer**: Transfer funds via Dwolla to other accounts 
- **Responsive**: Works seamlessly across desktop, tablet, and mobile 
 
## Getting Started 
 
**Prerequisites** 
- Node.js 
- npm 
 
"**Install dependencies**" 
```bash 
npm install 
``` 
 
"**Set up environment variables**" 
Create a `.env` file based on `.env.example` 
 
"**Run the app**" 
```bash 
npm run dev 
``` 
 
"Open http://localhost:3000"
